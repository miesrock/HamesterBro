# 经验球系统 SOP

1. CombatDirector 监听子弹击中敌人 (`_on_bullet_enemy_hit`)
2. 调用 `_spawn_experience_orb(hit_position)` 生成 ExperienceOrb
3. ExperienceOrb 设置 `experience_value`，记录到 "player" 组中的玩家
4. `_physics_process` 每帧检测与玩家距离
5. 进入 `attract_distance` 后 `is_attracting = true`
6. 吸引状态：朝玩家位置飞行，加速值可调
7. 到达 `collect_distance` 或 body_entered 碰撞玩家时调用 `_collect()`
8. `_collect()` 调用玩家 `add_experience(value)` 后 `queue_free()`
9. Player 累加 `experience` 变量（升级 UI 待对接）
