# 经验球/升级模板 SOP

## 适用场景

用于敌人死亡后掉落经验球，玩家靠近后自动吸附，收集经验并触发升级。

## 当前项目落点

- 经验球：`hamsterbroproject/scripts/experience_orb.gd`
- 生成位置：`hamsterbroproject/scripts/combat_director.gd`
- 玩家经验入口：`hamsterbroproject/scenes/player.gd`

## 当前已完成

1. 敌人被击中后生成 `ExperienceOrb`。
2. 经验球查找 `"player"` 组中的玩家。
3. 进入 `attract_distance` 后开始吸附。
4. 进入 `collect_distance` 或碰撞玩家后调用玩家 `add_experience(value)`。
5. 经验球收集后 `queue_free()`。

## 推荐参数

```gdscript
@export var experience_value: int = 1
@export var attract_speed: float = 12.0
@export var attract_distance: float = 4.0
@export var collect_distance: float = 0.55
@export var hover_height: float = 0.35
```

## 升级系统下一步

在 Player 中加入：

```gdscript
signal experience_changed(current: int, required: int)
signal level_up(new_level: int)
```

推荐字段：

```gdscript
var level: int = 1
var experience: int = 0
var experience_to_next_level: int = 10
```

推荐流程：

1. `add_experience(value)` 累加经验。
2. 经验达到阈值后扣除阈值。
3. `level += 1`。
4. 提高下一等级需求。
5. 发出 `level_up`。
6. UI 或升级菜单监听该 signal。

## 验收标准

- 敌人死亡后地上出现发光经验球。
- 玩家靠近时经验球会明显加速飞向玩家。
- 收集后经验球消失。
- 玩家经验值递增。
- 后续接 UI 时经验条能实时更新。

## 常见坑

- 经验球用 `Area3D` 时要确保有 `CollisionShape3D`。
- 经验球吸附时不要只追玩家脚底，建议加 `hover_height`，看起来更像飞向身体。
- 大量经验球会有性能压力，后续可做对象池或合并经验球。
