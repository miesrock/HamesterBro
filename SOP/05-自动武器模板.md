# 自动武器模板 SOP

## 适用场景

用于 Vampire Survivors 风格自动攻击：玩家不用瞄准，系统定时寻找目标并发射技能。

## 当前项目落点

- 自动武器控制：`hamsterbroproject/scripts/combat_director.gd`
- 子弹：`hamsterbroproject/scripts/bullet.gd`
- 枪口节点：`hamsterbroproject/scenes/Player.tscn` 的 `VisualPivot/Muzzle`

## 当前已完成

1. `CombatDirector` 持有 `player_path`。
2. 每隔 `fire_interval` 寻找最近敌人。
3. 按 `burst_count` 生成一组弹体。
4. 弹体有彩色发光材质。
5. 命中敌人后触发伤害字、火花、音效和经验球。

## 推荐参数

```gdscript
@export var fire_interval: float = 0.14
@export var burst_count: int = 3
@export var spread_degrees: float = 10.0
@export var muzzle_path: NodePath
```

## 实现步骤

1. 自动武器控制器放在主场景，命名 `CombatDirector`。
2. `_find_nearest_enemy()` 从 `"enemies"` 组中找最近目标。
3. 方向计算只用 XZ 平面，避免子弹向地面或天空飞。
4. 子弹出生点优先使用 `Muzzle.global_position`。
5. 如果没有 Muzzle，使用玩家位置 + 高度 + 方向偏移作为 fallback。
6. 子弹发出 `enemy_hit(enemy, hit_position)` signal。
7. `CombatDirector` 监听 signal 并生成反馈。

## 验收标准

- 有敌人时自动开火。
- 敌人越近越优先被攻击。
- 调整 `Muzzle` 节点位置后，子弹出生点随之改变。
- 调整 `burst_count` / `spread_degrees` 后，喷射感明显变化。

## 可扩展方向

- 增加武器资源 `WeaponData`：冷却、伤害、弹速、颜色、散射。
- 多武器同时运行：Hook、Orb、Cheese Gun。
- 增加对象池，避免高频 new/free 子弹。
- 增加命中次数、穿透、反弹、持续伤害。
