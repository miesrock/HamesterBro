# 程序化音频系统 SOP

1. JuicyAudio 继承 Node，通过 `AudioStreamWAV` 合成所有音效
2. `_build_streams()` 用采样函数生成 shoot/hit/crit/voice 四种流
3. `_make_stream(duration, sampler)` 按帧写入 16-bit PCM 数据
4. `_build_player_pool()` 创建 `pool_size` 个 AudioStreamPlayer 循环复用
5. 对外暴露方法：`play_shoot()` / `play_hit()` / `play_crit()` / `play_big_callout()`
6. CombatDirector 通过 `_play_audio("play_shoot")` 调用
7. Main.tscn 挂载 JuicyAudio 节点，CombatDirector 通过 `audio_path` 引用
